Tic-Tok-Toe

A browser-based Tic-Tok-Toe game built using HTML, CSS, and JavaScript, with a strong focus on DOM manipulation. Players take turns placing their marks, and the game automatically checks win conditions across rows, columns, and diagonals. The interface updates dynamically, offering a clean, responsive experience and a quick reset option for continuous play.

Guess My Number Game

A Java-based puzzle game built entirely using the Swing framework, where the player must discover a 4-digit secret number within 7 attempts. After every guess, the game shows two clues: Correct Digits: how many numbers are present in the hidden code Correct Positions: how many digits are both correct and in the right spot This combination of clue feedback encourages logical thinking and deduction. The interface uses input grids, color-coded feedback panels, and buttons for digit entry, creating a structured and interactive desktop gameplay experience.
